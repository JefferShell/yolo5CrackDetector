## 项目简介

这个系统主要为医生和专家提供一个平台，帮助医生遇到无法解决的患者问题时，将任务分派给专家并获取专业诊断和建议。

## 主要功能

**医生端：**

- 登录注册，完善用户信息
- 上传患者信息：基本信息和检查视频
- 上传视频时进行抽帧处理
- 指定专家对任务进行处理
- 查看专家诊断意见
- 个人中心：修改用户信息

**专家端：**

- 查看指定自己解决的任务
- 查看已完成的任务
- 指导对应任务：对抽帧的图片进行点击标注
- 提交指导意见并加载算法结果
- 修改自身指导的任务
- 个人中心：修改用户信息


## 技术架构

该系统采用以下技术架构：

* 前端：HTML、CSS、JavaScript
* 后端：Python、Django
* 数据存储：MySQL

## 使用说明

**医生端：**

1. 登录系统并完善用户信息。
2. 上传患者信息和检查视频。
3. 选择指定专家。
4. 查看专家诊断意见。

**专家端：**

1. 登录系统。
2. 查看指定自己解决的任务。
3. 查看已完成的任务。
4. 为任务提供指导，并提交自己的指导意见。
